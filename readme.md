# 📊 Ejemplos de Análisis

Durante el análisis se evaluaron diferentes factores que pueden influir en la cancelación de clientes, por ejemplo:

- tipo de contrato
- servicios adicionales contratados
- método de pago
- tiempo de permanencia del cliente
- costo mensual del servicio

Los modelos permiten identificar patrones que ayudan a estimar la probabilidad de churn.
